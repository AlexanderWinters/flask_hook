# Flask Hook
A flask webhook manager!

## For development
Clone project: 
```bash
git clone https://github.com/AlexanderWinters/flask_hook.git
```
Create new python environemnt and enable it:
```bash
python3 -m venv venv
source venv/bin/activate
```
Install dependencies:
```bash
pip install -r requirements.txt
```

Run app:
```bash
python webhook.py
```
